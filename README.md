## Projects

### 1. [Genomics Expression Analysis](./01-genomics-expression-analysis/)
**Tools:** Python, Pandas, scikit-learn, Matplotlib, Seaborn  
Differential expression analysis and ML classification on cancer genomics data. Includes feature selection, Random Forest classification, and biological interpretation of results.

### 2. [Drug Discovery ML Pipeline](./02-drug-discovery-ml/)
**Tools:** Python, RDKit, DeepChem, scikit-learn  
Predicting drug potency (pIC50) from molecular fingerprints using the ChEMBL database. Covers molecular featurization, model training, and chemical space visualization.

### 3. [Neuroimaging Classification](./03-neuroimaging-classification/)
**Tools:** Python, Nilearn, MNE-Python, scikit-learn  
Classifying cognitive states from brain imaging data. Feature extraction from fMRI/EEG data and ML-based classification with cross-validation.

### 4. [LLM Biology Benchmark](./04-llm-biology-benchmark/)
**Tools:** Python, Hugging Face Transformers, Claude API  
Evaluating how well large language models perform on biology reasoning tasks — gene function prediction, pathway analysis, and experimental design questions. Custom benchmark with scoring framework.

### 5. [LLM-Powered Biology Research Agent](./05-llm-bio-agent/)
**Tools:** Python, LangChain/Claude API, Biopython, NCBI Entrez  
An agentic workflow where an LLM queries biological databases (UniProt, NCBI, PDB) to answer research questions, retrieve gene annotations, and summarize literature.
