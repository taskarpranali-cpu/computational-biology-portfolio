# Python Foundations for Biology

## Overview

Foundational Python skills built from scratch using biology examples. These notebooks take a complete beginner through variables, data structures, file I/O, NumPy, and Pandas — all using DNA sequences, gene expression data, and real biology datasets.

## Notebooks (work through in order)

| Week | Notebook | Topics |
|------|----------|--------|
| 1 | `01_Week1_Python_for_Biology.ipynb` | Variables, lists, dictionaries, loops, functions, GC content analysis |
| 2 | `02_Week2_File_IO_for_Biology.ipynb` | String methods for DNA, FASTA parsing, CSV I/O, error handling, list comprehensions |
| 3 | `03_Week3_NumPy_Pandas_for_Biology.ipynb` | NumPy arrays, Pandas DataFrames, Breast Cancer dataset, missing data, merging datasets |

## Key Skills Demonstrated

- Python fundamentals applied to biological problems
- Parsing standard bioinformatics file formats (FASTA, CSV)
- Data manipulation with NumPy and Pandas
- Handling messy real-world data (missing values, type errors)
- Exploratory data analysis on the Breast Cancer Wisconsin dataset

## How to Run

```bash
pip install numpy pandas scikit-learn scipy matplotlib seaborn

jupyter notebook
```

Start with `01_Week1_Python_for_Biology.ipynb` and work through sequentially — each notebook builds on concepts from the previous one.
